Loading Bruker OPUS files
============================================

Here we load an experimental Bruker OPUS files and plot it.


.. code-block:: default


    import spectrochempy as scp

    Z = scp.read_opus(
        ["test.0000", "test.0001", "test.0002", "test.0003"], directory="irdata/OPUS"
    )
    print(Z)
